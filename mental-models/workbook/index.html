    How to Use This Workbook

    This workbook is designed to be filled in as you read the main module (Mental Models for AI-Assisted Development).

    Process: 1. Read each section of the main module 2. Answer the reflection questions in this workbook 3. Do the hands-on exercises 4. Complete the capstone exercise 5. Review your answers—you should be able to explain each model clearly

    Time: ~45 minutes (reading + exercises)
